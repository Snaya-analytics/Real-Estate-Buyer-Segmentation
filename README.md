# Real-Estate-Buyer-Segmentation
## 📑 Project Overview
This project analyzes real estate transactions to understand buyer segmentation, loan application behavior, and satisfaction scores.  
Data cleaning included ID normalization, sale price normalization, and removal of invalid entries.
## 📂 Dataset Description
- **buyer** → Unique buyer ID  
- **country, region** → Location details  
- **acquisition_purpose** → Reason for property purchase  
- **satisfaction_score** → Buyer satisfaction (numeric scale)  
- **loan_applied** → Whether loan was applied (Yes/No)  
- **transaction_date** → Date of transaction  
- **notes** → Data cleaning remarks
## 📂 Repository Contents
- `research_paper.ipynb` → Jupyter Notebook with analysis
- `buyers_final.csv` → Cleaned dataset
- `dashboard.py` → Streamlit dashboard code
## 📊 Features
- Buyer segmentation by region and acquisition purpose
- Loan application behavior analysis
- Satisfaction score visualization (heatmap, bar chart, pie chart, boxplot, line chart)
- Cleaned dataset with invalid rows dropped
